# PC Optimization Checklist

Run this safe checklist before trying advanced tweaks.

## Before you begin

- Back up important files.
- Create a restore point.
- Update Windows, GPU drivers, and chipset drivers.

## Quick wins

- Disable unnecessary startup apps.
- Uninstall software you no longer use.
- Clear temporary files.
- Keep at least 10-20% of SSD storage free.

## Stability checks

- Restart after major updates.
- Check Task Manager for high background usage.
- Avoid registry cleaners and mystery debloat scripts.

## Optional

- Use Performance mode when plugged in.
- Reduce animation effects if you prefer a snappier feel.
- Review browser extensions and remove anything unused.
